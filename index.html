03 — Projects

Selected work.

A mix of thesis research, coursework and applied collaborations — spanning deep learning, radar, and 3D point clouds.

Thesis · Deep Learning

Forest Disturbance Driver Classification

A year-long end-to-end project: processing raw forestry records, researching disturbance domains, and training a ResUNet model to classify forest disturbance drivers across EU forests using Sentinel-1/2 time series.

Sentinel-1/2 ResUNet PyTorch Python

Integration Course · Sweco & Gemeente Amsterdam

Subsurface Root Detection with GPR

Explored ground-penetrating radar to detect tree roots beneath Amsterdam streets, using point cloud clustering and root object reconstruction.

GPR Point Clouds CloudCompare

Independent Analysis · NDWI

Flood Mapping Across Central Europe

Produced NDWI-based flood visualizations from Copernicus data for the fall 2024 Central European floods and the 2021 Limburg floods, with a project group.

NDWI Copernicus Python

Advanced Remote Sensing · Groningen

3D Urban LiDAR Analysis

Built an interactive 3D map of building heights in Wageningen from DEM/DSM data, and used AHN LiDAR scans to assess change in urban tree greenery on Zernike Campus.

LiDAR (AHN) 3D Visualization Python

Energy Sector Customer Analysis
